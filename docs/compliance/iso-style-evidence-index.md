# Evidence Index (ISO-style)

This index points auditors to objective evidence generated by the CI pipeline.

## Build & release evidence
- Evidence Pack ZIP attached to each GitHub Release
- manifest.json (commit, tag, timestamp)

## Security evidence
- CodeQL results
- Dependency audit (npm + pip)
- Trivy filesystem scan
- SBOM

## Validation evidence
- KFRE pytest report
- KFRE validation report (dataset version + hash)
- Model card (versioned)

## Governance evidence
- PR list merged for release
- Approvers list
- Branch protection configuration (documented)
